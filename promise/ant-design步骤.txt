React使用：
在git里面搜索antd pro 找到14.7k的那个
git clone https://github.com/ant-design/ant-design-pro.git
克隆下来之后进行以下操作
cd ant-design-pro
npm install 或 yarn install
npm start 或 yarn start 运行之后会出来一个分析报告页面


Vue使用：
在git里面搜索vue-element-admin 进入一个git的页面
git clone git@github.com:PanJiaChen/vue-element-admin.git (起个name名比如: manage)
克隆下来之后进行以下操作
cd manage
cnpm install
npm run dev 运行之后显示出来一个分析页面就完成啦  
localhost:9527
